- IDE: [인텔리제이사용 (used Intellij)]
- DB: [MYSQL 사용]

## 주요 특징

- Spring Boot 프로젝트 기본 설정
- Spring Data JPA 예제
- JUnit을 이용한 테스트 코드 작성
- Thymeleaf를 이용한 화면 구성
- Spring Security를 이용한 회원 인증 및 인가 기능 구현

2. 상품 관리 기능

   - 상품 관리하기

3. 화면 구성
   - 메인 화면
   - 상품 상세 페이지

4. 주문 기능
   - 주문 기능 구현
   - 주문 이력 구현
   - 주문 취소하기
